# `Rofi`

[`Rofi`](https://github.com/DaveDavenport/rofi) 是一个窗口选择器，运行对话框，`ssh`启动器和 [`dmenu`](https://wiki.archlinux.org/title/Dmenu "Dmenu") 的替代品。    </br>

最初是由[`Sean Pringle`](https://github.com/seanpringle)编写，并由[`Dave Davenport`](https://github.com/DaveDavenport)扩展的[`simpleswitcher`](https://github.com/seanpringle/simpleswitcher)的克隆。    </br>

------

## 中文说明：

该仓库为本人正在使用的`Rofi`应用软件配置文件，    </br>

该仓库内配置文件主要参考了[`ArchWiki`中的`Rofi`配置说明](https://wiki.archlinux.org/title/Rofi_(%E7%AE%80%E4%BD%93%E4%B8%AD%E6%96%87)#配置)。    </br>

------

### 使用说明：

- ① 将该仓库克隆到`~/.config`文件夹内。    </br>

- ② 重启`Rofi`以应用该配置文件。    </br>

- ③ 即可使用该配置文件下的`Rofi`应用软件。    </br>

## English description：

[` Rofi `](https://github.com/DaveDavenport/rofi) is a window selector, an alternative to running dialog boxes, ` ssh ` launchers and [` dmenu `]((https://wiki.archlinux.org/title/Dmenu "Dmenu").    </br>
It was originally a clone of [` simpleswitch `](https://github.com/seanpringle/simpleswitcher) written by [` Sean Printer `](https://github.com/seanpringle) and extended by [` Dave Davenport `](https://github.com/DaveDavenport).    </br>

This warehouse is the configuration file of the `Rofi` application software I am using
